Autre truc: pour "patcher" un bug de montage CIFS dans la VM du dossier partagé sous Windows (via le script que je t'ai montré l'autre jour), il y'a une clé de registre à ajouter.
Voir le script cmd ci-dessous (il faut probablement lancer "cmd" en mode administrateur).

reg add HKLM\SYSTEM\CurrentControlSet\Services\LanmanServer\Parameters /v Size /t REG_DWORD /d 3 /f
